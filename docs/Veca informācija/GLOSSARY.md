# Project Glossary

Pēdējā atjaunināšana: 2026-04-07

## Kanoniskie termini

| Termins | Nozīme |
|---|---|
| HomeOS | Produkts kopumā |
| Module | Funkcionāla sadaļa (Kitchen, Finance, RESET, Events u.c.) |
| Household | Koplietojamā mājsaimniecības vienība |
| Profile | Lietotāja personīgais profils |
| Theme | Vizuālās pasaules konfigurācija |

## Tehniskie termini

| Termins | Nozīme |
|---|---|
| App Router | Next.js maršrutu modelis mapē `src/app` |
| Provider chain | React context secība no saknes layout |
| RLS | Row Level Security piekļuves kontrole DB slānī |
| BYOK | Bring Your Own Key; per-user AI atslēga |
| Vault | Supabase drošā server-side slepeno vērtību glabāšana |

## Domēna termini

| Termins | Nozīme |
|---|---|
| RESET | Privāts labsajūtas modulis |
| Seasonal override | Svētku virsuzslānis, kas ietekmē tēmas uzvedību |
| Bento dashboard | Sākuma ekrāna plīšu izkārtojums |
